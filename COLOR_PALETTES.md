## How to Change Your Palette

### Quick Change (CSS Variables)
Add this at the top of your `styles.css`:

```css
:root {
  /* Change these values to any palette above */
  --color-primary: #00ffff;
  --color-secondary: #ff00ff;
  --color-accent: #00ff00;
  --color-bg: #0a0a0a;
  --color-text: #e0e0e0;
}
```

Then replace hardcoded colors with:
- `#00ffff` ‚Üí `var(--color-primary)`
- `#ff00ff` ‚Üí `var(--color-secondary)`
- `#00ff00` ‚Üí `var(--color-accent)`

### Find & Replace Method
1. Open `styles.css`
2. Find: `#00ffff` Replace with: (your new primary color)
3. Find: `#ff00ff` Replace with: (your new secondary color)
4. Find: `#00ff00` Replace with: (your new accent color)

## Design Tips:

1. **Contrast is key:** Always ensure text is readable against backgrounds
2. **Limit colors:** Use 2-3 main colors max for consistency
3. **Test on mobile:** Colors look different on various screens
4. **Accessibility:** Check color contrast ratios (aim for 4.5:1 minimum)
5. **Brand consistency:** Use the same palette across all materials
